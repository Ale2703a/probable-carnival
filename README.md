Visão Geral do Projeto

Este **projeto é uma IA pessoal** em desenvolvimento, inspirada em assistentes inteligentes como *J.A.R.V.I.S. e na IA Shogun do Miyauti*.

O objetivo é criar um assistente capaz de interagir por texto e voz, executar comandos no computador e automatizar tarefas do sistema de forma progressiva.

🧠 Estado Atual do Projeto

O projeto foi iniciado e parcialmente desenvolvido com auxílio da IA (Claude Code), que ajudou na criação da estrutura inicial, organização do código e implementação de funcionalidades básicas.

Até o momento, já foram implementados:

Estrutura base do projeto
Sistema inicial de chat
Primeiras ideias de automação do sistema
Início de módulos de memória e organização de contexto
⚠️ Situação Atual

O desenvolvimento encontra-se temporariamente pausado devido ao meu nível atual de conhecimento em programação.

Apesar disso, o projeto continua como aprendizado ativo, sendo evoluído conforme o meu avanço nos estudos de desenvolvimento e inteligência artificial.

🎯 Objetivo Futuro

O objetivo é continuar evoluindo o sistema até que ele se torne um assistente pessoal completo, com capacidades como:

Execução de comandos no sistema operacional
Automação de tarefas no computador
Controle de navegador e aplicativos
Interação por voz (entrada e saída)
Memória contextual e aprendizado contínuo
🚀 Filosofia do Projeto

Este projeto não é apenas um software, mas um processo de aprendizado contínuo, onde cada etapa representa evolução prática em programação, automação e inteligência artificial.
